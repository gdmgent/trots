T.R.O.T.S
=========

Installation
------------

```
$ brew install ruby
$ gem install bundler
$ git clone https://github.com/gdmgent/trots
$ cd trots
$ bundle install
$ bundle update
$ bundle exec jekyll serve
```

Open
----

 - Local: <http://127.0.0.1:4000/trots/>
 - Online: <http://trots.gdm.gent>
